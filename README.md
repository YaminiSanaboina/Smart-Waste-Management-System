Smart Waste Management System
Overview
The Smart Waste Management System is an AI-powered application designed to classify waste into different categories such as plastic, paper, glass, metal, cardboard, and trash. This system aims to promote recycling awareness and contribute towards sustainable development by helping users understand proper waste segregation.
Key Features
•	Upload an image of waste for analysis
•	Automatic classification using Deep Learning models
•	Supports six waste categories: plastic, paper, glass, metal, cardboard, and trash
•	Provides environmental impact insights
•	Displays recycling tips for each category
•	Simple and user-friendly interface
Technologies Used
•	Python
•	TensorFlow / Keras
•	Streamlit
•	NumPy
•	OpenCV
Setup & Execution (VS Code)
1.	Open the project folder in Visual Studio Code
2.	Create a virtual environment: python -m venv tfenv
3.	Activate the environment: tfenv\Scripts\activate
4.	Install dependencies: pip install -r requirements.txt
5.	Run the application: streamlit run app.py
